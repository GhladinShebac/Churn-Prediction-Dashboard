ðŸ§  Key Features & Workflow
1. Data Engineering & RFM Analysis
Transformed 500,000+ raw transaction records into a customer-centric dataset by calculating RFM metrics:

Recency: Days since the last purchase.

Frequency: Total number of transactions.

Monetary: Total revenue generated by the customer.

2. Predictive Modeling
Defined "Churn" using a threshold based on Recency (Customers who haven't purchased in >60 days).

Trained an XGBoost Classifier to recognize patterns in buying behavior.

Engineered features such as Average Order Value (AOV) and Product Diversity to improve model accuracy from baseline.

3. Automated Retention Engine
Implemented a rule-based logic system that recommends actions based on the predicted risk:

VIP Recovery: Personal outreach + high-tier discounts for high-spending at-risk customers.

Loyalty Boost: Re-engagement campaigns for previously frequent shoppers.

Standard Nurture: Automated newsletter updates for low-risk active customers.
